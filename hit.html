# Hit

Each time the user interacts with the website to the iKala Data Lake, such as loading a web page on a website or loading a screen, clicking button, purchasing, etc.

In a mobile app, the tracking code is touched and the activity is noted. Each interaction is wrapped in a Hit and sent to the iKala Data Lake server.

Three type of hit：

  • Pageview hit
  • Event hit
  • Transaction hit
