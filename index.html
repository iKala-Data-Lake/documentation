# iKala Data Lake

# Overview

Different from the Data Warehouse, Data Lake can accommodate a large amount of data and is used as a storage pool for data materials for future business analysis and machine learning applications.

Unlike Google Analytics and the other DMP products, iKala Data Lake not only has users' behavior data of web/app, but also integrates user profiles, establishes a personalized recommendation system and classification/clustering modules to help us increase User experience and ROAS for advertising!

# How iKala Data Lake work?

iKala Data Lake provides a set of SDKs that let engineers put SDKs into your web/app, collect various behaviors that users use your web/app, and return to iKala Data Lake.

You can see the aggregated data in the backend(console) of iKala Data Lake as a dashboard, allowing you to see the data insight and make the right business strategy.
