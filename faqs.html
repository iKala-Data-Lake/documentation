# FAQs

Q: Why total count of「使用裝置分佈」is sometimes different from count of「新舊訪客分佈」?

A: When users use browser DevTool to change value of user-agent, that will make same Vistor ID associate with different devices. Then the count of「使用裝置分佈」is much than count of「新舊訪客分佈」.
