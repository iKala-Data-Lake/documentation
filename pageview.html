# Page View

Page view measurement allows you to measure the number of views you had for a particular page on your website/App.

This guide explains how to implement page tracking with iKala Data Lake.

# Web

When Javascript ika.init(); was executed. It will trigger pageview hit automatically.

Web pageview only be triggered once per page. so if you implement with Single-Page-Application web. You have to operate router manually to trigger pageview.

Following code shows How to implement page tracking by web:

# Web - Javascript

<script type="text/javascript" src="./dist/{debug or prod}/ika.js"></script>
<script type="text/javascript">
    ika.init("IKA-000000-1");
</script>

# iOS / Android

Developing iOS / Android pageview with iKala Data Lake. You need to set pageview command. Assign title and class parameters. to pageview function.class is like path of website ?

Following code shows How to implement page tracking by iOS / Android.

# Android - Java

tracker.setCurrentPage(MainActivity.class.getSimpleName(), "HomePage");

# iOS - Swift

try? Tracker.shared.setCurrentPage(className: "MyClassViewController", title: "Home Page")
